HTML

  • HTML is used to create the structure of a webpage
  • The title element is required and crucial for SEO
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Elements nested inside of other elements are called child elements

CSS

  • CSS is used to define how webpage content is presented to users
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • The period preceding "card" designates this as a class selector. Any element in the HTML file assigned to the class will gain these property assignments from the CSS
  • CSS declarations contain a property and a value for that property

Git

  • Git is a version control system
  • Use the command line to access Git
  • Git commannds always begin with git
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds modifications in the current branch to the staging area. -A flag indicates that we want to add all changes
  • git commit -m: commit staged content as a new commit snapshot, -m flag associates a message with our commit
  • Always do a git pull to make sure you have the latest version of your project on your local computer

JavaScript

  • JavaScript is a tool that takes a static HTML/CSS webpage and makes it interactive
  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • A function is a set of instructions that tells the computer how to perform a certain task
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to execute repeatedly
  • To use a for loop, you need the variable's index and the array's length
  • In order for the code in a function to run, you must call the function
